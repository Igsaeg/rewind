# 📸 Rewind

A digital memory album for our grade 7 to grade 10 years — built so we'd never forget those times.
<br>
## 📄 Tech Stack
- `React`
- `Vite`
- `JavaScript`
- `CSS`
- `Git`
- `Github Pages`

## ✨ Features
- **Organized Gallery**: Navigate through photos using a grade-based album system.
- **Optimized Loading** : Lazy loading keeps image browsing fast and efficient.
- **Immersive Viewer**: Open images in a full-screen modal for detailed viewing.
- **Theme Support**: Toggle between light and dark modes.
- **Responsive Layout**: A smooth experience across different screen sizes.

## 📁 Project Structure
- `src/` — React components and app logic
- `src/assets/` — images, SVG icons, and image metadata
- `src/content.jsx` — gallery content and lazy-loading behavior
- `src/header.jsx` — header, theme toggle, and info modal
- `src/sidebar.jsx` — grade selection navigation
